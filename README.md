# OTA 房价计算器

一个面向酒店运营人员的轻量在线房价计算网页，用于计算携程、美团、飞猪等 OTA 平台的后台卖价、外网展示价、平台佣金和酒店到手收入。

## 功能

- 支持携程、美团、飞猪三个平台。
- 支持正推：后台卖价 -> 外网展示价 -> 到手收入。
- 支持倒推外网价：目标外网价 -> 后台应录入卖价。
- 支持按到手反推：希望到手金额 -> 外网价 -> 后台应录入卖价。
- 支持综合活动折扣、佣金比例配置。
- 支持浏览器本地保存平台默认配置。
- 用户输入数字、切换平台或切换模式时实时计算。
- 默认结果四舍五入到整数，不展示括号内小数。

## 计算口径

第一版采用以下统一口径：

```text
外网展示价 = 后台卖价 * 综合活动折扣
佣金金额 = 外网展示价 * 佣金比例
到手收入 = 外网展示价 - 佣金金额
```

按希望到手金额反推时：

```text
外网展示价 = 希望到手金额 / (1 - 佣金比例)
后台卖价 = 外网展示价 / 综合活动折扣
```

## 本地使用

直接用浏览器打开 `index.html` 即可使用。

## GitHub Pages 部署

1. 将 `index.html`、`styles.css`、`app.js`、`README.md` 上传到 GitHub 仓库。
2. 进入仓库 `Settings`。
3. 打开 `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main` 或 `master`，目录选择 `/root`。
6. 保存后等待 GitHub 生成公开访问地址。

## 当前限制

- 不区分各 OTA 平台真实特殊规则。
- 不支持多层优惠券拆分，只支持一个综合活动折扣。
- 不支持 Excel 批量导入导出。
- 不保存云端历史记录。
